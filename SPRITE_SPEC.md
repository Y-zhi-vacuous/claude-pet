# 千千精灵帧规格说明

---

## 输出清单

为以下 11 个状态各生成 PNG 帧序列，放入对应文件夹：

```
assets/sprites/
├── idle/        frame_01~06.png   (6帧, 循环)
├── walkleft/    frame_01~04.png   (4帧, 循环)
├── walkright/   frame_01~04.png   (4帧, 循环)
├── walkup/      frame_01~04.png   (4帧, 循环)
├── walkdown/    frame_01~04.png   (4帧, 循环)
├── sitsleep/    frame_01~04.png   (4帧)
├── play/        frame_01~06.png   (6帧)
├── think/       frame_01~04.png   (4帧)
├── talk/        frame_01~04.png   (4帧, 循环)
├── happy/       frame_01~06.png   (6帧)
└── eat/         frame_01~04.png   (4帧)
```

**约 50 张 PNG，每张 96×96 像素，透明背景。**

---

## 技术规格

| 参数 | 值 |
|------|-----|
| 帧尺寸 | 96×96 px |
| 小狗主体占比 | 约 75×75 px，居中偏下（上方留空给耳朵/状态图标） |
| 格式 | PNG-24，透明背景 |
| 播放速度 | 150ms/帧（~6.7 fps），循环动画首尾无缝 |
| 缩放 | NearestNeighbor（点采样），不做抗锯齿 |
| 色深 | 建议控制在 12-16 色以内（纯像素风不需要渐变） |

## 调色板

```
暖白 #FFFAF5    主毛色（身体、头部、腿）
奶白 #F5EDE3    腹部/阴影面
米棕 #E8D5C4    身体暗面、腿内侧
耳棕 #B48C64    垂耳主色
深棕 #8B6914    耳根暗面
眼黑 #281E14    眼睛、鼻子
腮红 #FFB6B6    脸颊（开心时可见）
纯白 #FFFFFF    眼睛高光、毛尖
轮廓 #C4A882    极淡的轮廓线（可选，不要太重）
```

## 小狗特征速写

白色比熊犬：
- 圆形蓬松脑袋，顶部有微凸的绒毛
- 一对棕色垂耳，从头顶两侧垂下，耳长约头高的一半
- 黑色圆豆眼（2×2 像素），间距约 8-10px，有白色高光点
- 小三角形黑鼻子在脸部中央偏下
- 微笑嘴巴（一条弧线或两个像素点）
- 身体是椭圆形绒毛团，比头稍大
- 四条短腿，圆柱形，底部有小圆爪
- 尾巴是蓬松小毛球，翘在身体右上方
- 开心时脸颊会出现粉色腮红

## 逐状态详细描述

### idle（6帧, 循环播放）

小狗面朝右（或正面微侧），放松站立。

```
帧1: 正常站姿，身体高度=基准
帧2: 身体微微膨胀（scale 1.02），吸气
帧3: 身体继续膨胀（scale 1.03），达到最大
帧4: 身体开始收缩（scale 1.02），呼气
帧5: 恢复正常（scale 1.00）
帧6: 眨眼——眼睛变成一条横线（2×1 px），身体正常
```

关键：变化幅度很小（2-3px），呼吸感。尾巴每帧微微左右摆动 1-2px。

---

### walkleft（4帧, 循环播放）

小狗朝左走，身体略向左倾。

```
帧1: 左前腿前伸+右后腿后蹬 / 右前腿直立+左后腿直立
帧2: 左前腿直立+右后腿直立 / 右前腿前伸+左后腿后蹬
帧3: 镜像帧1的对角
帧4: 镜像帧2的对角
```

关键：身体有上下 1-2px 弹跳感，耳朵因走动微摆。尾巴随身体节奏摆动。

### walkright / walkup / walkdown

walkright = walkleft 的镜像翻转。
walkup：小狗背对镜头（看到圆屁股和尾巴），腿向前上方迈。
walkdown：小狗面对镜头（看到脸和肚子），腿向前下方迈。

---

### sitsleep（4帧）

小狗蜷成一团睡觉，面朝右或蜷成球。

```
帧1: 正常坐姿，闭眼，身体微缩
帧2: 身体进一步缩小(scale 0.95)，头靠在身体上
帧3: 最蜷缩状态(scale 0.92)，闭眼=两条细线
帧4: 微放松(scale 0.94)，重复呼吸循环
```

Zzz 标记：第 1、3 帧上方 2-3 个淡蓝色"z"飘出（可选，加分项）。

---

### play（6帧）

小狗侧滚玩耍。

```
帧1: 正常侧站
帧2: 身体开始向右倾斜 15°
帧3: 侧躺，腿朝上乱蹬
帧4: 身体向左倾斜 15°（滚回）
帧5: 接近正常，腿还在空中
帧6: 回到正常，微微摇晃
```

关键：翻滚轴心在身体中心，腿在帧 2-4 时离地乱摆。尾巴全程快速摇摆。

---

### think（4帧）

小狗歪头思考。

```
帧1: 正常站姿
帧2: 头向左歪 10°，眼睛微眯
帧3: 头向右歪 10°
帧4: 头回正
```

帧2-3 头顶上方出现一个闪烁的"?"（可选但加分）。

---

### talk（4帧, 循环）

小狗张嘴说话。

```
帧1: 闭嘴（正常嘴巴）
帧2: 嘴微张（嘴巴区域变大 1px 高）
帧3: 嘴张大（2px 高的黑色椭圆/方形）
帧4: 嘴微张（同帧2）
```

关键：只有嘴巴区域变化，其他部分不动。尾巴可以小幅摇摆增加生动感。

---

### happy（6帧）

小狗开心蹦跳，是动作幅度最大的状态。

```
帧1: 下蹲蓄力(scale 0.95, 身体下沉 3px)
帧2: 更低(scale 0.93, 下沉 4px)
帧3: 跳起！(身体上升 6px, scale 1.05, 耳朵飞起)
帧4: 空中最高点(上升 8px, 眯眯眼, 腮红可见)
帧5: 落地回弹(上升 2px, scale 1.02)
帧6: 站稳，尾巴大摇
```

关键：帧 3-4 耳朵向上飞起（离开头 3-4px），眼睛变成 ^^ 或眯成缝，脸颊出现粉色腮红。尾巴在帧 3-6 大幅度摇摆。

---

### eat（4帧）

小狗低头吃东西。

```
帧1: 正常站姿
帧2: 头低下 3px，身体微前倾
帧3: 头最低（低下 5px），嘴部微张
帧4: 抬头回正，嘴部咀嚼（微张）
```

关键：只有头部上下移动，身体基本不动。食物不需要画出来。

## ✅ 质量检查清单

- [ ] 每帧 96×96，透明背景，PNG 格式
- [ ] 像素风清晰可辨，无模糊/抗锯齿（NearestNeighbor）
- [ ] 小狗在帧内位置一致，不会跳动（除动画本身的位移外）
- [ ] 循环动画首帧和末帧能无缝衔接
- [ ] 比熊特征明显：白色绒毛、棕色垂耳、黑豆眼、圆脸、短腿
- [ ] 4 帧动画的动作幅度 < 6 帧动画
- [ ] 所有帧的主体（小狗）大小一致，光照方向一致
- [ ] 调色板颜色统一，没有渐变或半透明像素
