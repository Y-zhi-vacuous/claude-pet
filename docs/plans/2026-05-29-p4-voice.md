# P4: 声音阶段 — 语音引擎实现计划

> **Goal:** 说"嘿小狗"唤醒 → 说话 → 小狗转文字发给 Claude → Claude 回复朗读出来

**Architecture:** VoiceEngine 是独立线程的状态机，内部串联 IWakeWordDetector → VAD → ISTTEngine → ClaudeBridge → ITTSEngine。全部 opt-in（config.voiceEnabled）。接口化设计：IWakeWordDetector 有 Porcupine 实现 + 键盘热键 fallback；ISTTEngine 有 Azure/Whisper 两种实现；ITTSEngine 用 Edge TTS（免费）。

**Tech Stack:** C# 12, NAudio (录音+播放), Porcupine NuGet, Microsoft.CognitiveServices.Speech (可选)

**Prerequisite:** P1-P3 已完成
